# 🌐 Krishna Gupta - Personal Portfolio

Welcome to my personal portfolio website! This project is built using **HTML** and **CSS** to showcase my skills, projects, education, certifications, and contact information. It reflects my passion for web development and creating clean, responsive user interfaces.

## 🚀 Features

- ✨ Modern and responsive design
- 👨‍💻 About Me section
- 📂 Projects showcase
- 💡 Skills section
- 💼 Experience section
- 🎓 Education details
- 📜 Certifications
- 📞 Contact information
- 📄 Resume download button
- 📱 Mobile-friendly layout
- ⚡ Built using only HTML & CSS (No JavaScript)

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📁 Folder Structure

```
Portfolio/

```

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/Krishna-gupta73/Portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your favorite browser.
